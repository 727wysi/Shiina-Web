Commands You Can Use
<#if userPriv?seq_contains("NOMINATOR")>

Nominator Commands

!requests (aliases: !reqs): Check the nomination request queue.
!_map: Change the ranked status of the most recently /np'ed map.
<#if userPriv?seq_contains("MODERATOR")>

Moderator Commands

!notes: Retrieve logs for a specified player.
!addnote: Add a note to a specified player.
!silence: Silence a specified player with duration and reason.
!unsilence: Unsilence a specified player.
<#if userPriv?seq_contains("ADMINISTRATOR")>

Admin Commands

!user (aliases: !u): Show general information about a given user.
!restrict: Restrict a specified player's account with a reason.
!unrestrict: Unrestrict a specified player's account with a reason.
!alert: Send a notification to all players.
!alertuser (aliases: !alertu): Send a notification to a specified player.
!switchserv: Switch client internal endpoints to a specified IP.
!shutdown: Gracefully shut down the server.
<#if userPriv?seq_contains("TOURNEY_MANAGER")>

Mappool Management Commands

!pool_help (aliases: !h): Show all documented mappool commands accessible.
!pool_create (aliases: !c): Add a new mappool to the database.
!pool_delete (aliases: !del, !d): Remove a mappool from the database.
!pool_add (aliases: !a): Add a new map to a mappool in the database.
!pool_remove (aliases: !rm, !r): Remove a map from a mappool in the database.
!pool_list (aliases: !l): List all existing mappools.
!pool_info (aliases: !i): Retrieve all information for a specific mappool.
<#if userPriv?seq_contains("DEVELOPER")>

Developer Commands

!stealth: Toggle developer stealth mode.
!recalc: Recalculate pp for a given map or all maps.
!addpriv: Grant specific privileges to a player.
!rmpriv: Remove specific privileges from a player.
!givedonator: Grant donator status to a player for a set duration.
!wipemap: Wipe scores from a map.
!server: Show performance data about the server.
